# ランニングアプリ・アシスタント

このスキルは、ランニングアプリのコード作成、デバッグ、および機能改善を支援するためのガイドラインを提供します。

## 開発の原則
- **モバイルファースト**: 屋外での使用を想定し、モバイルフレンドリーなUI/UXを最優先する
- **オフライン対応**: PWAとしての機能を維持し、電波の悪い場所でも動作するようにする（Service Worker, Manifest）
- **データの永続化**: ローカルストレージ（localStorage）を使用したデータの保存と同期
- **コード品質**: バニラJavaScript/CSSを使用し、シンプルでメンテナブルなコードを維持する

## 実装上の重要なルール
- **カスタムモーダル**: `confirm()` などのブラウザ標準ダイアログは避け、アプリ独自のカスタムモーダル（`showCustomConfirm` 等）を使用する
- **ハンバーガーメニュー**: モバイルでのナビゲーションはハンバーガーメニューに集約し、クリッカブルな範囲と z-index に注意する
- **目標管理**: 月間目標の進捗をプログレスバーで可視化し、達成時にはアニメーション（紙吹雪など）で演出する
